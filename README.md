# GM Bot in Go!

This is a simple Discord bot written in Go that simply replies to any user of the discord who says "gm" and will track a user's amount of gm's they have said!

## Deployment

To deploy this project run

```bash
  go build
```

Then simply run 

```bash
  ./gm-bot -t <bot-token>
```
